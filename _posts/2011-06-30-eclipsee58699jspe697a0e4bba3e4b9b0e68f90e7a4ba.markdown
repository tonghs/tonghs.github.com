有的朋友在用eclipse写jsp代码的时候，对正常的java代码有提示，而敲入像out、request等对象时没有代码提示，这是因为项目中没有添加Server Runtime的库。  
在项目的Properties -> Java Build Path -> Libraries -> Add Library -> Server Runtime里选入所需的库（需要配置服务器，我的是tomcat服务器）即可。
