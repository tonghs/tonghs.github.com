**某日在配置vista的IIS7时发现了一个问题，提示如下：**





**“An error occurred on the
server when processing the URL. Please contact the system
administrator”**





**百度一下，解决方法如下：**






**打开IIS7的asp设置，展开“调试属性”选项，“将错误发送到浏览器”这项默认的是False，改为True，然后点右侧的应用！**
